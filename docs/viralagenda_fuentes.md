## Estructura de URLs

```
https://www.viralagenda.com/es/{ccaa}/{provincia}/{zona}
```

Ejemplos:
- `/es/andalucia/granada/granada` → Ciudad de Granada
- `/es/andalucia/granada/alpujarra-granadina` → Comarca Alpujarra
- `/es/galicia/a-coruna/santiago` → Santiago de Compostela

## Selectores CSS

```css
/* Event cards */
li.viral-event

/* Title */
.viral-event-title a

/* Date */
.viral-event-date  /* Formato: JUE05FEBHOY o JUE05FEBHOYHASTA28FEB */

/* Location (hora|ciudad|venue|categoria concatenados) */
.viral-event-places

/* Category */
.viral-event-cats a
```

## Formato de Fechas

- Simple: `JUE05FEB` → 5 de febrero
- Con "HOY": `JUE05FEBHOY` → 5 de febrero (es hoy)
- Rango: `JUE05FEBHOYHASTA28FEB` → 5 al 28 de febrero

Meses: ENE, FEB, MAR, ABR, MAY, JUN, JUL, AGO, SEP, OCT, NOV, DIC
